# Mario Console Game (C++)

This project is a Mario-inspired console-based game developed in C++. It simulates basic platformer gameplay using a text-based interface, allowing players to control a character, avoid obstacles, and score points. The project focuses on implementing core programming concepts without the use of external graphics libraries.

## Overview

The game provides a simple yet interactive experience where the player navigates through a console-rendered environment. It demonstrates fundamental game development techniques such as game loops, collision detection, keyboard input handling, and score tracking.

## Features

- Console-based Mario-style gameplay  
- Real-time player movement  
- Obstacle interaction and collision detection  
- Score tracking system  
- Continuous game loop for dynamic gameplay  
- Simple and intuitive controls  

## Technologies Used

- C++
- Standard libraries (iostream, conio.h, etc.)
- Console-based rendering

## How the Game Works

### Game Loop
The game runs in a continuous loop that:
- Updates the game state  
- Accepts player input  
- Renders the updated frame in the console  

### Player Movement
The player can move the character using keyboard inputs to navigate through the game environment.

### Obstacles
Obstacles appear in the path of the player. The player must avoid collisions to continue playing.

### Scoring
Points are awarded based on survival time or successful navigation through obstacles.

### Game Over
The game ends when the player collides with an obstacle.
